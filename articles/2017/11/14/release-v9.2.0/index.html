  • 작성자: Evan Lucas
  • 원문: Node v9.2.0 (Current)
  • 번역: freenice12

주요 변경사항

  • crypto:
    • OpenSSL 1.1.0과 1.0.2 모두 지원합니다. (David Benjamin) #16130
  • fs:
    • 이제 fs.realpathSync.native와 fs.realpath.native 함수를 사용 할 수 있습니다. (Ben Noordhuis) #15776
  • process:
    • 이제 process.ppid 함수를 사용 할 수 있습니다. (cjihrig) #16839
